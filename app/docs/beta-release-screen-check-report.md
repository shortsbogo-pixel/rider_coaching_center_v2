# 베타 공개 전 실제 화면 점검 리포트

## 목적

라이더 코칭센터 Ver.2 1차 베타 공개 직전에 실제 Vercel 배포 URL 기준으로 화면 점검 결과를 기록하기 위한 리포트 양식이다.

이번 문서는 점검 기록용이다. 기능 코드, DB/Supabase 연결, 외부 API, 문자, GPS, 실시간 콜, 실제 음악 재생, Vercel/GitHub 배포 설정은 추가하거나 변경하지 않는다. 실제 개인정보, 원천 엑셀 파일, 정산 원본 파일도 포함하지 않는다.

## 점검 후 보관 원칙

- 실제 개인정보가 포함된 캡처는 문서에 붙이지 않는다.
- 원천 엑셀 파일이나 정산 원본 파일은 문서에 첨부하지 않는다.
- 오류 캡처가 필요하면 라이더명, 전화번호, 주소, 정산 민감정보를 가린다.
- 공개 보류 사유와 보완 조치 예정일은 반드시 기록한다.
