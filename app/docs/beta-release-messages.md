# 베타 공개용 안내문

## 목적

라이더 코칭센터 Ver.2 1차 베타 테스트를 실제 관리자와 라이더에게 공유할 때 사용할 짧은 안내문을 정리한다.

이번 문서는 공지/안내문 작성용이다. 문자 발송 기능, DB/Supabase 연결, 외부 API 연결, Vercel/GitHub 배포 설정, 기능 코드 로직은 추가하거나 변경하지 않는다.

## 주요 주소

- 라이더 코칭센터: https://rider-coaching-center-v2.vercel.app
- 라이더 정산앱: https://ridersettle-fmyozrbw.manus.space/

## A. 관리자용 카카오톡 / 문자 안내문

```text
[라이더 코칭센터 Ver.2 1차 베타 테스트 안내]

접속 주소
https://rider-coaching-center-v2.vercel.app

관리자 테스트 계정
ID: admin
PW: admin1234

확인할 메뉴
대시보드 / 업로드 / 검수 / 코칭 / 더보기

확인 요청
1. 엑셀 업로드 화면에서 파일 선택과 미리보기가 되는지
2. 검수 숫자와 이슈 안내가 이해되는지
3. “이번 주차 데이터로 반영” 후 대시보드/코칭 화면이 바뀌는지
4. 코칭 메시지 수정, 노출 ON/OFF, 내부 메모가 보이는지
5. 더보기에서 운영 로그와 정산앱 링크가 보이는지

주의
이 앱은 실시간 관제 앱이 아니라 주차별 확정 엑셀 데이터 기반 코칭 앱입니다.
정산 계산, 지급, 확정은 기존 정산앱에서 확인합니다.

테스트 중 오류나 헷갈리는 문구가 있으면 화면 캡처와 함께 알려주세요.
```

## B. 라이더용 카카오톡 / 문자 안내문

```text
[라이더 코칭센터 Ver.2 1차 베타 테스트 안내]

접속 주소
https://rider-coaching-center-v2.vercel.app

라이더 테스트 계정
ID: rider1
PW: rider1234

확인할 메뉴
내현황 / 내오더 / 내지도 / 내코칭 / MY

확인 요청
1. 내현황에서 오늘의 페이스 체크가 이해되는지
2. 오늘 완료 콜 수, 컨디션, 식사, 수면, 휴식을 입력해보기
3. 이번 주 목표와 추천 행동 문구가 이해되는지
4. 내오더, 내지도, 내코칭이 잘 보이는지
5. MY 화면에서 정산앱 링크가 잘 열리는지

주의
이 앱은 실시간 콜 진행 앱이 아닙니다.
관리자가 올린 최신 업로드 주차 기준으로 코칭 정보를 보여주는 앱입니다.
오늘의 페이스 체크 입력값은 이 기기 기준 임시 저장입니다.
정산 상세와 지급 상태는 기존 정산앱에서 확인합니다.

사용하면서 불편한 점이나 이해 안 되는 문구가 있으면 캡처와 함께 알려주세요.
```

## C. 승인대기 계정 테스트 안내문

```text
[승인대기 계정 테스트 안내]

접속 주소
https://rider-coaching-center-v2.vercel.app

승인대기 테스트 계정
ID: pending
PW: pending1234

확인 요청
1. 로그인 후 승인대기 화면만 보이는지 확인
2. 관리자 화면에 들어갈 수 없는지 확인
3. 라이더 데이터 화면에 들어갈 수 없는지 확인
4. 내현황, 내오더, 내지도, 내코칭, MY 데이터가 보이지 않는지 확인

승인대기 계정은 데이터 접근이 막혀 있어야 정상입니다.
다른 화면이 보이면 캡처해서 알려주세요.
```

## D. 내부 운영자용 짧은 공유문

```text
[오늘 베타 테스트 진행 순서]

1. 관리자 계정으로 접속
2. 대시보드 확인
3. 엑셀 업로드 미리보기 확인
4. 이번 주차 데이터로 반영
5. 검수 / 코칭 / 운영 로그 확인
6. 라이더 계정으로 접속
7. 내현황 / 오늘의 페이스 체크 / 내오더 / 내지도 / 내코칭 / MY 확인
8. pending 계정으로 승인대기 화면 확인
9. 코칭센터 ↔ 정산앱 링크 왕복 확인

오류 발생 시
- 화면 캡처
- 어떤 계정으로 접속했는지
- 어떤 메뉴에서 발생했는지
- 무엇을 누른 뒤 발생했는지
- 기대한 동작이 무엇인지
를 같이 남겨주세요.

피드백 분류
- 긴급: 로그인 불가, 화면 진입 불가, 업로드 불가, 잘못된 권한 노출
- 높음: 주요 수치 오류, 본인 외 데이터 노출, 정산앱 링크 실패
- 보통: 문구 혼란, UI 불편, 일부 화면 깨짐
- 낮음: 색상/표현 선호, 추가 아이디어

테스트 완료 후 확인 문서
- app/docs/beta-release-final-checklist.md
- app/docs/beta-operations-guide.md
- app/docs/beta-feedback-form.md
- app/docs/settlement-coaching-link-policy.md
```

## 공유 전 확인 문구

베타 참여자에게 안내문을 보내기 전에 아래 표현이 포함되어 있는지 확인한다.

- 주차별 확정 데이터 기반 코칭 앱
- 실시간 콜 진행 앱이 아님
- 정산 상세는 기존 정산앱에서 확인
- 오늘의 페이스 체크 입력은 이 기기 기준 임시 저장
- 오류나 불편사항은 캡처와 함께 피드백 요청
