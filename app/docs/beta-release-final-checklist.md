# 베타 공개 전 최종 운영 체크리스트

## 목적

라이더 코칭센터 Ver.2를 1차 베타 참여자에게 공개하기 전에 운영자가 마지막으로 확인해야 할 항목을 정리한다.

이번 문서는 운영 점검 문서다. 기능 코드, DB/Supabase 연결, 외부 API, 문자 발송, GPS/현재 위치, 실시간 콜 연동, 실제 음악 재생, Vercel/GitHub 배포 설정은 변경하지 않는다. 실제 개인정보, 원천 엑셀 파일, 정산 원본 파일도 추가하지 않는다.

- 코칭센터는 정산 계산 또는 정산 엑셀 파싱을 하지 않는다.

## 최종 공개 전 한 줄 원칙

1. 코칭센터는 쿠팡플러스 주차별 확정 엑셀 데이터 기반 코칭 도구다.
2. 코칭센터는 실시간 관제 앱이 아니다.
3. 코칭센터는 정산앱을 대체하지 않는다.
4. 원천 엑셀 파일과 실제 개인정보는 코드/배포물/피드백 문서에 포함하지 않는다.
5. 베타 피드백은 오류와 기능 추가 요청을 분리해 수집한다.
